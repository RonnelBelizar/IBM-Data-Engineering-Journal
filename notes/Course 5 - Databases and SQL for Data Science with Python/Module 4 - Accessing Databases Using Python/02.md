## ðŸŽ¯ Matplotlib

Matplotlib is Pythonâ€™s **foundational plotting library** â€” powerful, flexible, and perfect for customizing visualizations.

| Feature            | Description                                                         |
|--------------------|---------------------------------------------------------------------|
| Basic Plots        | Create line, bar, scatter, and pie charts.                          |
| Customizable       | Control colors, labels, axes, and figure size.                      |
| Works with NumPy   | Great for numerical arrays and time series data.                    |
| Integrates Easily  | Works smoothly with Pandas DataFrames.                              |

**Example (String Form):**  
"import matplotlib.pyplot as plt"  
"x = [1, 2, 3, 4, 5]"  
"y = [10, 20, 25, 30, 35]"  
"plt.plot(x, y, color='blue', marker='o')"  
"plt.title('Sample Line Chart')"  
"plt.xlabel('X-Axis')"  
"plt.ylabel('Y-Axis')"  
"plt.show()"

**Output:**  
A simple line chart showing the relationship between X and Y values.

## ðŸŒˆ Seaborn

Seaborn is built **on top of Matplotlib** â€” it makes charts **prettier, simpler**, and perfect for **statistical data**.

| Feature                | Description                                                 |
|------------------------|-------------------------------------------------------------|
| Beautiful Themes       | Automatically styles charts with clean color palettes.      |
| Easy Data Integration  | Works directly with Pandas DataFrames.                      |
| Statistical Insights   | Supports correlation plots, regression lines, and histograms.|
| Variety of Plots       | Includes boxplots, heatmaps, pairplots, barplots, etc.      |

**Example (String Form):**  
"import seaborn as sns"  
"import pandas as pd"  
"data = {'age': [23, 45, 31, 35, 52, 46], 'salary': [2500, 4000, 3200, 3600, 4800, 4200]}"  
"df = pd.DataFrame(data)"  
"sns.scatterplot(x='age', y='salary', data=df)"  

**Output:**  
A clean, colorful scatter plot showing the relationship between age and salary.
